# cerebral.work

Landing site for [cerebral.work](https://cerebral.work) — neuroscience-grounded tools for LLM coding harnesses.

Featured projects:
- **[reverie](https://github.com/cerebral-work/reverie)** — memory consolidation daemon
- **meshctl** — 7-role orchestration CLI + TUI (bundled with reverie)

## Stack
Astro 6 + MDX + Tailwind v4 · Bun runtime · Cloudflare Pages.

## Dev
```
bun install
bun run dev
bun run build
```
